Supports .yaml and .yml files
📖 Supported YAML Formats

Single Recipe:

recipe:
  title: "Pasta Carbonara"
  estimated_cooking_time: "20 minutes"
  serves: 4
  ingredients:
    - name: "Pasta"
      qty: "400g"
  steps:
    - step: "Cook pasta"

Multiple Recipes (List Format):

recipes:
  - recipe:
      title: "Recipe 1"
      # ... recipe details
  - recipe:
      title: "Recipe 2"
      # ... recipe details

Multiple Recipes (Document Separator):

recipe:
  title: "Recipe 1"
  # ... recipe details
---
recipe:
  title: "Recipe 2"
  # ... recipe details
